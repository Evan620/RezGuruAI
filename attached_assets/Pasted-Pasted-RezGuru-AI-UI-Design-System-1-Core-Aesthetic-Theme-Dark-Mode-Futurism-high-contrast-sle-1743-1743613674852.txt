🎨 RezGuru AI - UI Design System
1. Core Aesthetic
Theme: "Dark Mode Futurism" (high contrast, sleek, data-focused)

Color Palette:

Primary: #6E56CF (violet, for CTAs)

Secondary: #1A1A2E (dark navy, backgrounds)

Accent: #00F5D4 (teal, highlights)

Text: #F8F9FA (light gray, readability)

Fonts:

Headings: Geist Bold (techy yet clean)

Body: Inter Medium (high readability)

📱 Key Screens & Components
1. Dashboard (Homepage)
Dashboard

Top Bar:

Quick stats (e.g., 42 New Leads | $1.2M Pipeline)

AI Assistant toggle (floating chat bubble)

Kanban Board:

Columns: 🔥 Hot Leads (Score >80) | 📞 Contacted | 💰 Closed

Cards show:

Profile pic (generated via DiceBear)

Motivation score (radial progress bar)

Last action (AI Called 2h ago)

Right Sidebar:

Real-time lead map (Leaflet.js + OpenStreetMap)

AI Insights panel ("Top 3 ZIPs today: 85004, 90210")

2. Automation Studio
Automation Studio

Left Panel: Triggers (drag onto canvas)

New Lead | Email Opened | Form Submitted

Canvas: Node-based workflow builder (like n8n)

Nodes glow #00F5D4 when active

Connector arrows animate on hover

Right Panel: Action configurator

Example SMS action:

plaintext
Copy
Hi {{name}}, we buy homes in {{city}}! 
Reply STOP to opt out.
3. Lead Profile Page
Lead Profile

Header:

Name + tags (Probate, Urgent)

Big motivation score (0-100 dial)

Timeline:

📞 AI Call: 2024-05-20 14:30

📧 Email Sent: "Quick Cash Offer"

Documents Section:

Auto-generated contracts (PDF preview thumbnail)

One-click "Sign with DocuSign" button

4. AI Chat Interface
AI Chat

Personality: "Guru" (wise but approachable avatar)

Features:

Voice toggle (Azure TTS speaks responses)

Suggested prompts:
"Draft a contract for 123 Main St"
"Call this lead at 2 PM"

Context Awareness: Remembers past interactions

🛠️ Implementation Guide
1. Frontend Libraries
bash
Copy
npm install @radix-ui/react-dialog @radix-ui/react-dropdown-menu framer-motion react-query
Animations: Framer Motion

Charts: Visx (for data viz)

Tables: TanStack Table + column sorting

2. Sample React Component (Kanban Card)
tsx
Copy
interface LeadCardProps {
  name: string;
  score: number;
  lastContact: string;
}

const LeadCard = ({ name, score, lastContact }: LeadCardProps) => (
  <div className="bg-[#1A1A2E] rounded-lg p-4 border-l-4 border-[#6E56CF]">
    <div className="flex items-center gap-3">
      <img 
        src={`https://api.dicebear.com/7.x/pixel-art/svg?seed=${name}`} 
        className="w-10 h-10 rounded-full"
      />
      <div>
        <h3 className="text-[#F8F9FA] font-medium">{name}</h3>
        <div className="flex items-center gap-2">
          <CircularProgressBar value={score} /> {/* Score 0-100 */}
          <span className="text-[#00F5D4] text-sm">{lastContact}</span>
        </div>
      </div>
    </div>
  </div>
);
3. Micro-Interactions
Hover Effects: Cards scale transform: scale(1.02)

Loading States: Skeleton screens with #2A2A3A pulse animation

Notifications: Toast messages slide in from bottom-right

💡 Pro Design Tips
Prioritize Density:

Use compact tables (7px padding) to show more data

AI Personality:

Give the AI avatar a name ("Guru") and quirky tooltips

Error States:

Scrape failed? Show a "🛸 UFO retrieving data" Lottie animation
